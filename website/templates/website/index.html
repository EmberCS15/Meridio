About Merídio


Merídio(Greek-Share) is a free Filehosting platform where users can host their own repositories of files/videos/images etc and thus make it available for download by any of the registered members of the website.The key aim is to create a network of users who can host items in their repositories and allow other people to access the repository contents and also download stuff from others repositories.

Merídio also has an inbuilt chat feature where you can add friends via requests and chat with your friends.

Thats about it. So register and Merídio :).
